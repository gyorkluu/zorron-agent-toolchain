# Skill: pdf-reader

## 描述
读取和解析 PDF 文件内容，提取文本、表格和元数据。

## 触发条件
- 当用户请求读取、分析或提取 PDF 文件内容时自动激活
- 当用户提到 PDF 文件处理相关任务时

## 指令
1. 使用合适的工具打开 PDF 文件
2. 提取文本内容，保持原始格式和结构
3. 如果包含表格，以结构化格式呈现
4. 提取文档元数据（标题、作者、页数等）
5. 对提取的内容进行摘要或分析（根据用户请求）

## 输出格式
- 文本内容以 Markdown 格式输出
- 表格以 Markdown 表格格式呈现
- 元数据以键值对列表呈现

## 注意事项
- 扫描版 PDF 可能需要 OCR 处理
- 加密 PDF 需要用户提供密码
- 大文件应分段处理，避免内存溢出
