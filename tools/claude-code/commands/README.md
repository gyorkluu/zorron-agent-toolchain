# Claude Code 自定义命令

## deploy
部署 Zorron Agent Toolchain 配置到当前环境。
运行 `./install.sh` 来检测已安装工具并部署对应配置。

## status
查看当前 Zorron Agent Toolchain 的部署状态。
运行 `./scripts/zorron status` 来查看详细信息。

## add-skill
添加新的 Skill 到共享配置。
用法: `./scripts/zorron add skill <名称>`

## add-mcp
添加新的 MCP 服务。
用法: `./scripts/zorron add mcp <名称>`
